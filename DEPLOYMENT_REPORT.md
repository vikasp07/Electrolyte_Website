## 🚀 How to Use

### Clone the Repository
```bash
git clone https://github.com/vikasp07/Electrolyte_Website.git
cd Electrolyte_Website
```

### Run Frontend
```bash
cd frontend
npm install
npm start
```

The app will start at `http://localhost:3000`

### Build for Production
```bash
cd frontend
npm run build
```
